## Installation

#### Clone the Repository
```bash
git clone https://github.com/WesleyReboucas/memoteca.git
cd memoteca/
npm i
```

#### Server
```bash
cd memoteca/server
npm start
```

#### Project
```bash
cd memoteca/
npm start
```
